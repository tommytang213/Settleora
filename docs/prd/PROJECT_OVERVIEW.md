# Settleora - Project Overview

## Working Product Name
**Settleora**

This is the current working product name for the project. It should still be treated as a working name until final legal/trademark/domain/store clearance is done.

## Product Summary
Settleora is a self-hosted cross-platform expense management, shared bill assignment, settlement tracking, receipt OCR, recurring bill, forecasting, and reconciliation platform.

It is intended to support:
- personal expense recording
- shared expense and bill assignment
- settlement and payment tracking
- recurring and scheduled future bills
- OCR-based receipt ingestion with item-level assignment
- local-only mode
- server-connected mode with sync
- iOS, Android, user web, and admin web experiences
- strong security defaults with flexible deployment and policy controls
- automated build, test, packaging, and release workflows
